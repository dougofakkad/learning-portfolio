Flame Treatment Module

This module was created to address a training requirement at Arrival involving the preparation of panels via flame treatment. The module structure was closely aligned to the task, with interactive checkpoints for learners to drive home key points. I created animations for each task in Adobe After Effects from images prepared in Illustrator, as well as storyboarding individual sections.

Attributes module

Developed for a requirement to educate users of specific software, this module used learning checks to assess the need for users to review each section or to simply use it as a refresher if desired. I devised the module concept, storyboarded each section and developed the animations in Illustrator and After Effects.

Service, Maintenance & Repair training

This is an example of a structure created in HTML/CSS to present content required to educate learners in the SMR (Service, Maintenance, Repair) aspects of each vehicle system. The requirement was for three sections with text supported by either a 3D model, video animation, or image sequences. The 3D models were implemented using Google's Model Viewer API.

ADAS calibration module

This module was created for Thatcham Research as a refresher on ADAS systems and an introduction to calibration. I described the basic ADAS functions using animations created in Adobe Animate, and the calibration procedure as a video tutorial with checkpoints to reinforce key information, which I filmed and edited.

This is also an example of a redesign I completed on all Thatcham's e-learning modules. The look and feel was derived from then-current branding and the central idea was to move away from the 'slideshow' paradigm common in e-learning by having each progression as part of an interaction, and as such hopefully immerse the learner a bit more.

Animations

Animations created to illustrate various automotive functions and parts. As Arrival was producing an in-house vehicle, many of these could use CAD data to create specific and accurate representations. For the 2D animations, I rendered images from CAD models and used Photoshop to prepare them as assets for animation in After Effects. For 3D animation, I used Keyshot to apply materials, render and animate parts.
